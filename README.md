## üöÄ Project Overview
This website is a premium digital experience designed to market Apple's MacBook Pro. It goes beyond static images by integrating real-time 3D rendering and synchronized scroll animations to communicate performance, AI features, and design aesthetics.

## ‚ú® Key Features

### üßä Interactive 3D Visualization
- **Real-time Product Viewer**: Interactive 3D models of the 14" and 16" MacBook Pro using **React Three Fiber**.
- **Customization Engine**: Toggle between Space Gray and Dark finishes with real-time texture switching.
- **Dynamic Scaling**: Seamlessly switch between screen sizes (14" vs 16") with model re-scaling.
- **Studio Lighting**: Professional 3D lighting setup for high-fidelity model visibility.

### üé≠ Advanced Animations
- **ScrollTrigger Experiences**: GSAP-powered animations that transform models, fade text, and scale video content as the user scrolls.
- **AI Feature Showcase**: 5 AI-powered capability reveals (Email AI, Image AI, etc.) synchronized with scrollable feature videos.
- **Performance Reveals**: Animated spec reveals and performance image staging.

### üì± Responsive & Optimized
- **Mobile-First Design**: Adaptive layouts using **Tailwind CSS 4.0**.
- **Performance Aware**: Heavy 3D animations are intelligently adjusted or disabled on mobile devices to ensure smooth performance.
- **State Management**: Centralized store via **Zustand** for synchronized model properties (color, scale, texture).

## üìÇ Website Structure
1. **Hero Section**: High-impact branding with background video and pricing CTA.
2. **Product Viewer**: Interactive 3D model with color and size selectors.
3. **Showcase**: Gaming/performance video pinned with GSAP scale effects.

5. **AI Features**: Scroll-driven video boxes highlighting new AI writing and image tools.
